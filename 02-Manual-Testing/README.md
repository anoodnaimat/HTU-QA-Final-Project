## Manual Testing

### Overview

This section covers the manual testing activities performed on the Swag Labs web application.
The objective is to validate core system functionalities through manual execution of test cases
and to document execution results, defects, and supporting evidence.

---

### Application Under Test

- Application Name: Swag Labs
- Application Type: Web Application
- Testing Type: Manual Functional Testing

---

### Test Coverage

The manual testing covers the following scenarios:

- Valid login functionality
- Invalid login handling
- Adding products to the cart
- Removing products from the cart
- Sorting products using different sorting options

Both positive and negative test scenarios were executed.

---

### Test Documentation

Manual testing documentation is maintained in a single Excel file that contains two sheets:

#### Sheet 1: Test Cases and Test Execution Results

Each test case includes:

- Test Case ID
- Test Title and Description
- Test Steps
- Test Data
- Expected Result
- Actual Result
- Test Status (Pass / Fail)
- Execution Evidence (Screenshots)

#### Sheet 2: Bug Report

Each reported bug includes:

- Bug ID
- Bug Description
- Steps to Reproduce
- Expected Result
- Actual Result
- Severity / Priority
- Bug Status
- Screen recording (video) evidence

---

### Test Execution

- All test cases were executed manually.
- Test execution results are documented in the Test Cases sheet.
- Each test case includes its execution status and related evidence.

---

### Defect Reporting

- Defects identified during test execution are documented in the Bug Report sheet.
- Screen recordings (videos) are provided to clearly demonstrate reported issues.

---

### Evidence

- Evidence is provided for both test execution and reported defects.
- Evidence includes:
  - Screenshots for manual test case execution
  - Screen recordings (videos) for reported bugs
- Evidence links are included within the Excel file.

---

### Files Included

- Manual Testing Excel file:
  - Sheet 1: Test Cases & Execution Results
  - Sheet 2: Bug Report
- Screenshots (Test Execution Evidence)
- Screen Recordings (Bug Evidence)

---

### Notes

- Manual testing focuses on core functional scenarios only.
- Mobile application testing is not included.
